How the Confidence Score Works

Each result is scored from 0–100 based on the evidence Claude found. The score tells you how certain the tool is that the handle returned is the brand's best official account. Scores below 60 should be manually reviewed before use.

Handle Selection Priority

1
Linked from the brand's official website

If a handle is directly linked in the footer, header, or contact page of the brand's own website, that handle is selected. This is the strongest possible signal.

2
Global handle preferred when website-linked AND higher follower count

If both a global and a regional account exist, the global handle is preferred only if it is linked to or from the brand's official website AND has a higher follower count than the regional variant. Both conditions must be true.

3
Highest follower count

If no handle is linked from the official website, or the global handle doesn't meet Priority 2 conditions, the account with the most followers is selected — whether global or regional.

4
Other verification signals

If follower counts are unavailable: bio links to the official website, cross-source confirmation, handle name match, and verified badge are used to decide.

5
Manual review flag

If the tool cannot confidently choose between candidates, it sets "Yes" in the Manual Review column and notes the reason.

The profile's bio includes a link that points to the brand's official website, confirming the two are connected.

Confirmed across multiple sources

The same handle appears in multiple independent places — such as the brand's website, Wikipedia, press articles, or other directories — reducing the risk of a single misleading source.

+20
High follower count

The profile's follower count is consistent with an established brand of this type and size — suggesting it is the main, active account rather than an abandoned or minor one.

+18
Handle matches brand name

The handle is an exact or near-exact match to the brand name, reducing the chance of confusion with fan pages or unrelated accounts.

+15
Verified badge

The profile has a blue checkmark / verified badge, indicating the platform has confirmed it as an authentic account.

−10
Found via web search only

The handle was not found on the brand's official website and was located only through a general web search. Results from web searches are harder to verify as official.

−15
Multiple competing accounts found

Several accounts exist that could plausibly be the brand's official handle. Even after applying the selection rules, some ambiguity remains — the result should be reviewed.

−10
Follower count not verified

The tool could not confirm the follower count for the selected handle, making it harder to compare candidates or gauge account size.

−5
Regional variant selected (global account also exists)

A regional sub-account (e.g. nike_vietnam) was selected even though a global account also exists — typically because the regional one had more followers. This penalty does NOT apply if the regional account is the only one found.
